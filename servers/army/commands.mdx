## 概要

軍隊サーバーでは、軍隊を作ったり土地を保護するためにはコマンドを駆使する必要があります。大きく分けて以下のコマンドが用意されています。

- /army 軍隊の作成や設定など、軍隊関連の操作をするコマンドです。
- /armychat 軍隊メンバー間チャットが可能です。
- /market プレイヤーマーケットを開きます。
- /world 資源ワールドに移動するためのコマンドです。
- /vote 投票URLを表示します。
- /money 所持金を表示します。
- /pay 他のプレイヤーにお金を送金します。

## /army コマンド

軍隊の作成や設定など、軍隊関連の操作をするコマンドです。\
エイリアスとして`/a`,`/f`を使用できます。

**・情報コマンド**

| コマンド                               | 説明                                       | 権限     |
| :--------------------------------- | :--------------------------------------- | :----- |
| /army list                         | 存在する軍隊の一覧を表示します。                         | 全プレイヤー |
| /army info \<軍隊のシンボル\>             | 軍隊の情報を表示します。シンボルを指定した場合その軍隊の情報を表示します。    | 全プレイヤー |
| /army create \<軍隊のシンボル\> \<軍隊の名前\> | 自分の軍隊を作成します。                             | 全プレイヤー |
| /army delete                       | 自分の軍隊を削除します。                             | 最高指揮官  |
| /army set \<種類\> \<内容\>            | 自分の軍隊の設定をします。名前、シンボル、説明、色、公開/非公開を設定できます。 | オーナー   |

**・人事コマンド**

軍隊に入隊したり、プレイヤーを招待したりできます。

| コマンド                    | 説明                    | 権限     |
| :---------------------- | :-------------------- | :----- |
| /army join \<軍隊のシンボル\>  | 指定した軍隊に入隊します。         | 全プレイヤー |
| /army leave             | 軍隊から脱退します。            | 全プレイヤー |
| /army invite \<プレイヤー名\> | 指定したプレイヤーを軍隊に招待します。   | 指揮官    |
| /army accept            | 招待を受けている場合、招待を受け入れます。 | 全プレイヤー |
| /army reject            | 招待を受けている場合、招待を拒否します。  | 全プレイヤー |
| /army kick \<プレイヤー名\>   | プレイヤーを軍隊から追放します。      | 指揮官    |

**・領土**

領土を操作します。

| コマンド             | 説明                        | 権限   |
| :--------------- | :------------------------ | :--- |
| /army claim      | チャンクを取得し、領土として保護します。      | オーナー |
| /army unclaim    | チャンクを手放します。               | オーナー |
| /army unclaimall | 全てのチャンクを手放します。            | オーナー |
| /army buy_chunk  | 軍隊資金で土地購入します。領土上限を拡張できます。 | オーナー |

**・関係性**

軍隊間の関係性を定義します。これはほぼ、意思表示をするためのものです。

| コマンド                     | 説明                   | 権限   |
| :----------------------- | :------------------- | :--- |
| /army ally \<軍隊シンボル\>    | 指定した軍隊を同盟として定義します。   | オーナー |
| /army enemy \<軍隊シンボル\>   | 指定した軍隊を敵対として定義します。   | オーナー |
| /army neutral \<軍隊シンボル\> | 指定した軍隊との関係性定義を解除します。 | オーナー |

**・ホーム設定**

軍隊のホーム座標を設定すると、テレポートできます。

| コマンド          | 説明                | 権限    |
| :------------ | :---------------- | :---- |
| /army home    | 軍隊ホーム座標にテレポートします。 | 全メンバー |
| /army sethome | 軍隊ホーム座標を設定します。    | オーナー  |

**・銀行**

銀行への入出金を操作します。入金は全メンバーが行えますが、出金は指揮官のみです。

| コマンド                  | 説明               | 権限    |
| --------------------- | ---------------- | ----- |
| /army withdraw \<金額\> | 銀行から指定した金額を出金します | 指揮官   |
| /army deposit \<金額\>  | 銀行へ指定した金額を入金します  | 全メンバー |

## /armychat　コマンド

軍隊チャットができます。軍隊内で作戦会議などをしたい場合などに活用できます。

| コマンド                    | 短縮系 | 説明           |
| :---------------------- | --- | :----------- |
| /armychat \<発言テキスト...\> | /ac | 軍隊チャットで発言します |

## /market　コマンド

マーケットUIを開きます。マーケットでは、他プレイヤーとアイテムの売買を行えます。\
エイリアスとして`/m`を使用できます。

<Frame>
  ![Army Market](/images/army-market.png)
</Frame>

| コマンド    | エイリアス | 説明            |
| :------ | ----- | :------------ |
| /market | /m    | マーケットUIを開きます。 |

## /world　コマンド

`/world`ワールド移動するためのコマンドです。ワールドを移動するためには5秒間動かずに待機する必要があります。安全な場所で実行しましょう！

<Tip>
  /army home コマンドも同様に、軍隊のホーム座標へテレポートできます。
</Tip>

| コマンド         | 短縮   | 説明               |
| :----------- | ---- | :--------------- |
| /world main  | /w m | メインワールドへテレポートします |
| /world sigen | /w s | 資源ワールドへテレポートします  |

## /vote　投票URL表示

投票すると、サーバー内でデイリークエストを完了でき、お金がもらえます。\
ログイン状態で投票しないと貰えないことがあるので注意！

| コマンド  | 説明          |
| :---- | :---------- |
| /vote | 投票URLを表示します |

## /money　所持金確認

現在あなたが所持しているお金を表示します。

| コマンド   | 説明                 |
| ------ | ------------------ |
| /money | あなたが所持しているお金を表示します |

## /pay　送金

あなたは所持金を他のプレイヤーに直接送金できます。

| コマンド                   | 説明            |
| ---------------------- | ------------- |
| /pay \<プレイヤー名\> \<金額\> | 他のプレイヤーに送金します |
