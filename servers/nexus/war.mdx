# 戦争の流れ

軍隊サーバーでは、ネクサスへの侵攻を宣戦布告することで戦争ができます。\
5分間の準備時間のあと、戦争が開始されます。

<Steps>
  <Step title="宣戦布告をする">
    ```宣戦布告
    /nexus attack <NexusID>
    ```

    ５分間の準備時間のあと、戦争が開始されます。
  </Step>
  <Step title="戦争開始">
    ネクサスのある場所にいき、ネクサスにダメージを与えることでネクサスの体力を削ることが出来ます。敵プレイヤーが防衛にくる可能性があるので、攻撃にも備えて下さい！
  </Step>
  <Step title="戦争の終結">
    ネクサスの体力を削り切ったら、ネクサス争奪戦争に勝利し、ネクサスは自分の軍隊の所有物になります。
  </Step>
</Steps>

# 和解金と軍隊体力

戦争を始めるには同じ金額の和解金を設定する必要があります。和解金は戦争が開始されたとき、軍隊体力として扱われます。軍隊体力が底を尽きると勝敗が決定します。

和解金が少ないと、すぐに決着がつきます。\
和解金が多い場合、戦争は長期にわたるでしょう。

## 和解金

和解金は宣戦布告時に決めます。和解金で設定された金額が軍隊銀行にある必要があります。\
宣戦布告した場合、すぐに和解金が軍隊銀行から引き出され、勝敗がきまるまで戻されません。

## 軍隊体力

相手の軍隊拠点に RPG-7 を打ち込みブロックにダメージを与えることで、相手の防衛資金を削ることができます。なお、ブロックは破壊されても自動修復されるため、チェストの中身や原型は保護されています。

# 勝敗の決まり方

戦争中はPVPが許可され、お互いの銃のダメージが通るようになります。敵軍隊プレイヤーをキルしたり敵軍隊拠点に RPG-7 を打ち込むことで、相手軍隊の体力を削りきりましょう。

<u>軍隊体力が0になったら</u>負けです。（ほぼ同時に０になった場合は引き分けとなります）

# 戦争コマンド

| コマンド                                              | 説明                    | 備考         | 権限   |
| :------------------------------------------------ | :-------------------- | :--------- | :--- |
| /army declare \<敵軍隊のシンボル\> \<和解金\> \<ブロック破壊ありなし\> | 和解金を指定して、敵軍隊に宣戦布告します。 |            | オーナー |
| /army declarecancel \<敵軍隊のシンボル\>                  | 宣戦布告をキャンセルします。        |            | オーナー |
| /army surrender \<敵軍隊のシンボル\>                      | 指定して敵軍隊との戦争に降伏します。    | 戦争中のみ可能です。 | オーナー |

# Q&A

### 戦争で勝つとどんなメリットがありますか？

戦争で勝利すると、和解金を獲得できます。

### 戦争で負けるとどのようなデメリットがありますか？

和解金を奪い取られます。

### 戦争時のブロック破壊をOFFにできますか？

宣戦布告コマンドを実行する際の引数で、`<ブロック破壊ありなし>` を `off` に設定してしてください。\
お互いの合意の上で、ブロック破壊の有無を決めて戦争できます。

### 戦争しない場合でも拠点を破壊されたり、チェストからアイテムをぬすまれたりしますか？

戦争をしない場合、拠点は常にブロック設置・破壊から保護されています。ブロックはRPG-7 の爆撃からも保護されており、破壊されることはありません。ただし、１枚壁など脆弱な場合、村人や動物は<u>RPG-7などの爆発ダメージを受ける場合があります</u>。領土を広げ、<u>地上や空からアクセスできないよう厳重な防御</u>をしてください。

また、チェストのアイテムは戦争中でも敵軍にぬすまれることはありません。チェストにアクセスできるのは、軍隊のメンバーのみです。

戦争が好みでない場合は、安全にこの世界で暮らしていくことができます。

### 戦争で爆破されたチェストはどうなりますか？

自動補修システムにより、時間が経つと中身そのままで復活します。
