## サーバーの概要

名前の通り、基本的にKITを選んで戦うPVPです。

## KITの選択

手に何も持たずにしゃがむ（スニークをする）とメニュー（現在バグでプレイヤーの位置に出現しますが、ページを更新すると修正されます）が開けるので、そこから選ぶか、ロビーの墓石から選ぶかの二通りあります。キルストリークアイテム(三連続キルすると入手できる強力な武器)やサブアイテムの種類は現在メニューでのみ選択できるので、メニューがおすすめです。

## 試合の始めかた

中央から前に進むとあるボタンを押すと開始できますが、条件があります。\
試合を開始するためには <u>２人以上の準備完了者が必要</u> です。

緑色の<u>参加可能の看板</u>を右クリックすると試合に参加する状態になり、<u>準備完了の看板</u>を押すと準備完了状態になります。また、赤色の<u>不参加看板</u>を右クリックすると試合に参加しない状態になり、準備完了状態が解除されます。

## どんなモードがあるの?

現在以下の5モードがあります。

| 種類                  | 内容                                                       |
| ------------------- | -------------------------------------------------------- |
| FFA                 | シンプルなFFA。Free For All。1vs1vs1vs...vs1                    |
| Team \| Red vs Blue | シンプルな2チーム戦                                               |
| Team \| 3point      | 2チームに分かれて戦う。ラウンドごとに分かれ、3ラウンド先取。                          |
| FFA \| CHAOS        | キルストリークアイテムのみで戦うモード。カオス。                                 |
| FFA \| Random       | KIT、サブアイテム、キルストリークアイテムすべてがランダムで決まるモード。死亡時にまたランダム。カオス。    |
| Team \| Conquest    | 3つのポイントを2チーム間で占領し合い、先にポイントを0まで減らした方の勝ち。(自分のポイントが0になれば勝ち) |
